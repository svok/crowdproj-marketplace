## Инструкции для устранения дефектов

### Блокирующие (должны быть исправлены обязательно)

1. **ОГРН:** Добавить в таблицу полей ИЛИ явно указать причину исключения (например, сноску «ОГРН не отображается на экране карточки, так как не публикуется в публичных API согласно BR-005 — поле доступно только для бизнес-логики и внутренних сервисов»).
2. **ContactEmail / ContactPhone:** Добавить примечание к полям: «Отображается только для администратора предприятия (не публикуется публично, BR-005)».
3. **Доп. действия:** Синхронизировать с screen-flows — либо добавить эти действия в screen-flows (и переутвердить), либо удалить из screen-2.

### Рекомендуемые

4. **Проверить, является ли Экран 2 публичным или только для администратора.** От этого зависит, какие поля должны отображаться. Если экран административный — указать это в секции «Назначение экрана».
